# Контрольная работа №1
В скриншотах в директории `report/` выполняются стандартные команды, но с помощью псевдонимов. Вот все псевдонимы и их значение:
- `mcr` - `micro` (CLI-редактор файлов)
- `gm` - `git merge`
- `gchk` - `git checkout`
- `gcl` - `git clone`
- `ga` - `git add`
- `gcmsg <message>` - `git commit -m <message>`
- `push` - `git push`
